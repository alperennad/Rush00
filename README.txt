Bu proje Ecole42 okullarının ilk hafta sonu grup projesidir.Sizden kullanıcıdan alınan sayıya uygun şekil oluşturan bir program yazmanız beklenmektedir. Benim kodum bütün hepsine uyduğundan sadece bazı harf ve sekilleri değiştirerek butun egzersizler ile uyumlu olmasını saglayabilirsiniz. Main fonksiyonunu bonusu almak icin argümanlı bir sekilde yaptım. Argüman konusu C06 konusu olmasina rağmen bakmanızı öneririm.

This project is the first weekend group project of the Ecole42 schools. You are expected to write a program that creates a shape according to the number entered by the user. Since my code works for all of them, you can make it compatible with all exercises by simply changing some letters and shapes. I made the main function in an argumented way to get the bonus. Although the argument topic is a C06 topic, I recommend you to look at it.
